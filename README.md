# conways-game-of-life-c
This project is a port/copy of the famous Conway's Game of Life created in C as a part of a passion project.
![image alt](https://github.com/XENON-github/conways-game-of-life-c/blob/2e3e747a79fec095d0f78c2268d2634f7b7fb112/media/Screenshot_20251227_112745.png)

## Installation

Clone the repository and build the project using `make`:

```bash
# Clone the repository
git clone https://github.com/XENON-github/conways-game-of-life-c
# Navigate into the project directory
cd conways-game-of-life-c
# Build the project
make
# Install the executable
make install

# To run the program
conway
